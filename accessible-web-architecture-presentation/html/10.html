So What? It's Just a Bunch of Angle Brackets

HTML markup is...

  • Plain old text
  • Easily parsed by any programming language
  • Ubiquitous: can be parsed by other programs, not just web browsers
  • Meaningful across platforms
  • Stable and referenceable
  • (Relatively) understandable by humans
  • Tolerant of mistakes

Programming code is...

  • Usually tied to a platform
  • Requires complex runtimes and parsers
  • Opaque: once compiled, code can no longer be easily understood or parsed by other programs
