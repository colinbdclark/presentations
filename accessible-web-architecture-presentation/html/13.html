What Does It All Mean?

  • The way something on the web looks is changeable
  • Browsers, authors, other programs, and users are able to influence how something is presented
  • A single structure can be presented in myriad ways
  • The structure of the web is easily understood
  • Other programs can parse, manipulate, and represent the user interface
  • The web is naturally cross-platform, cross-device, and cross-presentation
